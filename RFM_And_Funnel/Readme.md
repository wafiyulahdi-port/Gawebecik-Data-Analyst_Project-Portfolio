This project implements an end-to-end RFM (Recency, Frequency, Monetary) segmentation pipeline for end users using transactional sales data stored in MySQL. The script extracts, cleans, and filters customer purchase data (including return-behavior exclusion), calculates RFM metrics per customer and product, and applies K-Means clustering to classify customers into segments such as Champion, Loyal, Potential Loyal, and Hibernate. The processed data is then automatically uploaded to a MySQL table as part of an automated ETL workflow, with logging and error handling included for production use.
